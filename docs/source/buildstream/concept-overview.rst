What is BuildStreaM?
====================

Omnia BuildStream provides a comprehensive automation solution for managing infrastructure build workflows. It uses a catalog-driven approach where you define your build requirements in structured catalog files, and BuildStreaM executes automated pipelines to create and deploy images according to your specifications.

BuildStreaM addresses key challenges in HPC cluster image management:

   - **Automation**: Eliminates manual build and deployment processes
   - **Integration**: Works seamlessly with existing Omnia deployments
   - **Traceability**: Provides complete audit trails for all build operations
